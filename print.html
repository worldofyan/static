Introduction

mdBook is a command line tool to create books with Markdown. It is ideal for creating product or API documentation, tutorials, course materials or anything that requires a clean, easily navigable and customizable presentation.

  • Lightweight Markdown syntax helps you focus more on your content
  • Integrated search support
  • Color syntax highlighting for code blocks for many different languages
  • Theme files allow customizing the formatting of the output
  • Preprocessors can provide extensions for custom syntax and modifying content
  • Backends can render the output to multiple formats
  • Written in Rust for speed, safety, and simplicity
  • Automated testing of Rust code samples

This guide is an example of what mdBook produces. mdBook is used by the Rust programming language project, and The Rust Programming Language book is another fine example of mdBook in action.

Contributing

mdBook is free and open source. You can find the source code on GitHub and issues and feature requests can be posted on the GitHub issue tracker. mdBook relies on the community to fix bugs and add features: if you'd like to contribute, please read the CONTRIBUTING guide and consider opening a pull request.

Se déplacer

Se déplacer d'un caractère

  • l (à droite de k) se déplace vers la droite.
  • h (à gauche de j) se déplace vers la gauche.
  • fx déplace le curseur sur l'occurrence suivante du caractère x dans la ligne.
  • Fx déplace le curseur sur l'occurrence précédente du caractère x dans la ligne.
  • tx déplace le curseur sur le caractère précédent l'occurrence du caractère x dans la ligne.
  • Tx déplace le curseur sur le caractère suivant l'occurrence du caractère x dans la ligne.

Se déplacer au début d'un mot

  • w (word) vers la droite. Note : certains caractères spéciaux comptent comme des mots (ex : *blabla* comptent comme trois mots).
  • W (Word) vers la droite. Note : les caractères spéciaux comptent avec le mot accroché (ex : *blabla* compte comme un mot).
  • b (backward) vers la gauche. Note : certains caractères spéciaux comptent comme des mots (ex : *blabla* comptent comme trois mots).
  • B (Backward) vers la gauche. Note : les caractères spéciaux comptent avec le mot accroché (ex : *blabla* compte comme un mot).

Se déplacer en fin de mot

  • e (end) vers la droite. Comme pour w mais en fin de mot.
  • E (End) vers la droite. Comme pour W mais en fin de mot.

Se déplacer en début ou fin de ligne

  • $ se déplace en fin de ligne.
  • ^ se déplace en sur le premier caractère non blanc d'une ligne.
  • + ou ↵ (touche plus ou entrée) saute une ligne et se déplace sur le premier caractère non blanc d'une ligne.
  • - se met sur la ligne au dessus et se déplace sur le premier caractère non blanc.
  • 0 (zéro) se déplace en début de ligne (les caractères blancs ne changent rien).

Se déplacer dans un endroit précis d'une ligne

  • n| se déplace sur le caractère correspondant à la colonne ne de la ligne. Si n est trop grand, va en fin de ligne.

Se déplacer parmi les lignes

  • j ou ↓ se déplace d'une ligne en dessous.
  • k se déplace d'une ligne au dessus.
  • gg se déplace tout en début de texte.
  • G se déplace tout à la fin du texte.
  • nG se déplace à la ligne n du texte. On peut afficher le numéro de la ligne où on se trouve avec la commande :nu (number). Même chose avec la commande CTRL-G qui affiche aussi le pourcentage de texte au dessus du curseur et le numéro de colonne.

Se déplacer dans une phrase

  • ( déplace le curseur au début de la phrase en cours (la phrase d'avant se finit avec les caractères suivants : ?, ., !.
  • ) déplace le curseur au début de la phrase suivante (la phrase suivante est précédée par les caractères suivants : ?, ., !.

Se déplacer dans un paragraphe

  • { déplace le curseur au début du paragraphe courant (le paragraphe d'avant finit par une ligne blanche).
  • } déplace le curseur au début du paragraphe suivant (le paragraphe suivant est précédé par une ligne blanche).

Multiplier les déplacements

On peut ajouter un nombre devant chaque commande de déplacement(sauf 0). Par exemple 2w provoque le déplacement du curseur de 2 mots.

Se déplacer dans l'écran

  • H déplace le curseur en haut de l'écran visible.
  • nH déplace le curseur en haut de l'écran et descend de n lignes en comptant la première.
  • M déplace le curseur au milieu de l'écran visible.
  • L déplace le curseur en bas de l'écran visible.
  • nL déplace le curseur en bas de l'écran visible et remonte de n lignes en comptant la dernière.

Déplacer l'écran (scrolling)

Note : pour toutes les commandes si dessous, essayer les lettres en minuscule si ça ne fonctionne pas (ex : CTRL-f à la place de CTRL-F).

  • CTRL-F (Forward)(essayer sans majuscule si ça ne fonctionne pas) fait défiler d'un écran vers le bas.
  • CTRL-B (Backward) fait défiler d'un écran vers le haut.
  • CTRL-D (Down) fait défiler d'un demi écran vers le bas.
  • CTRL-U (Up) fait défiler d'un demi écran vers le haut.
  • CTRL-E fait défiler l'écran d'une ligne d'écran vers le haut.
  • CTRL-Y fait défiler l'écran d'une ligne d'écran vers le bas.
  • z↵ (appuyer sur z puis la touche entrée) fait défiler la ligne courante vers le haut de l'écran sans bouger le curseur (le curseur se retrouve en haut de l'écran).
  • z. fait défiler la ligne courante vers le centre de l'écran sans bouger le curseur (le curseur se retrouve en milieu d'écran).
  • z- fait défiler la ligne courante vers le bas de l'écran sans bouger le curseur.

Insérer du texte

Commande Insert

i (insert = insérer) permet d'insérer du texte à gauche du curseur. Il faudra appuyer sur ESC pour sortir du mode --insert--. Le mode insertion correspond à une édition dans les éditeurs de texte classiques.

Commande Append

a (append = ajouter) permet d'insérer du texte à droite du curseur. Il faudra appuyer sur ESC pour sortir du mode --insert-- (l'appui sur a fait passer en mode insertion de texte).

Remplacer du texte

Remplacer un caractère

  • r (replace = remplacer) permet de remplacer un caractère sans passer en mode insertion.
  • R (Replace = Remplacer) permet de se mettre en mode overstrike, c'est à dire que le curseur remplace tout ce qui suit jusqu'en fin de ligne si on saisi du texte (un caractère ou plus). Dans ce cas on passe en mode -- REPLACE -- jusqu'à l'appui sur la touche ESC.

Remplacer un mot

  • cw (change word = changer mot) commence par effacer le mot à partir de la position du curseur (jusqu'à la fin du mot) puis permet son remplacement. Un appui sur w permet de se placer en début de mot avant son remplacement entier.
  • c3w comme cw mais permet le remplacement de trois mot par le nombre de caractères ou mots que l'on veut.
